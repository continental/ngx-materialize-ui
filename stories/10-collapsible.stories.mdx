# Collapsible
The **Collapsible** component is intendet to show overview data with detail data on the same page.

## Basic
In the most basic version, the collapsible can open and close sections.
<Preview>
  <Story name='Basic' height='250px'>{{
    template: `
      <mui-root-container>
        <mui-collapsible>
          <li>

            <div class="collapsible-body"><span>Lorem ipsum dolor sit amet.</span></div>
          </li>
          <li>
            <div class="collapsible-header"><i class="material-icons">place</i>Second</div>

          </li>
        </mui-collapsible>
      </mui-root-container>`
  }}
  </Story>
</Preview>
