# Select
The **Select** form control can be used to let the user choose one value out of a set of given options.

## Basic
In the most basic version, the Select just shows a label by setting the `label` attribute. You can set the
available select options by setting the `options` attribute to an array of objects conaining a `value` and 
a `label` attribute by default. If label and value have different key use the `optionsValueKey` and the 
`optionsLabelKey` to customise for your individual key names. The Select is also compatible withe the 
default `ngModel` directive used in template driven forms.
