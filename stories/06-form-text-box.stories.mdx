# Text Box
The **Text Box** form control can be used to show a single line text input.

## Basic
In the most basic version, the Text Box just shows a label by setting the `label` attribute. The **most important** step when using 
the Text Box is to set a view unique name to the `name` attribute. The Text Box is also compatible withe the default `ngModel`
directive used in template driven forms.
