# Text Area
The **Text Area** form control can be used to input, show and edit multiple lines of text.

## Basic
In the most basic version, the Text Area just shows a label by setting the `label` attribute. The **most important** step when using 
the Text Area is to set a unique name to the `name` attribute. The Text Area is also compatible withe the default `ngModel`
directive used in template driven forms.
